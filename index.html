Project K3

Project K3 is a web application aimed to give teachers the ability to offer courses online with abilities to teach, assess, and track students.

The application is inspired by Khan Academy and Udacity. However, this, when completed, will allow anyone to post lessons of their own, effectively bringing these tools to the masses. Teachers everywhere could use this software for free to teach their students.

The project is currently in private alpha. I'm testing it with a grade 11 physics class and a programming course.
