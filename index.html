Python

Python is a great programming language! There's nothing bad that I could say about the language, really.

This is my primary programming language as I'm most experienced in it. In fact, this website's generator is written in Python + Flask.

Also, another language that I've beginning to like very much is GNU Octave. Great modeling language, great substitute for matplotlib for prototyping.

Though, we all need to move to Python 3.x!

FRC

FRC is the FIRST Robotics Competition. This competition is a high school level robot competition that has more than 45000 students building robots to accomplish a certain task.

I am currently involved with the program as the captain and lead programmer of Team 4069 (Lo-Ellen Robotics).

This year, we are working on a basketball shooter. My main goal is to get a reasonably accurate auto shooter programmed in Java.
